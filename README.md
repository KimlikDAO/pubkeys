# Açık anahtarlar

KimlikAŞ'ye ait 1024 adet Ed25519 açık anahtarı. Repo şu an `testing.pubkeys` dosyasında test anahtarlarını içeriyor; asıl anahtarlar ileriki bir tarihte
yayımlanacak.

Toplamda yaklaşık 200M tane imza atacağı için her gizli anahtar `200.000±500`
kere kullanılacak. Bu rakam Ed25519 için tamamen güvenli sınırlar içinde.
